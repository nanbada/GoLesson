# 01. PRD — 제품 요구사항

버전 v1.0 (2026-07-03) · 모든 요구사항은 ID / Requirement / Acceptance / Priority(M=Must, S=Should, C=Could) 형식.

## 1. Vision

초등 영어/수학 1:1 학원을 위한 최소 기능 관리 도구.
**5초 안에 수업 시작, 30초 안에 기록 완료.** 강사의 기록 시간을 줄이고 학부모 소통 품질을 올린다.

## 2. Goals

학생 조회(GoAlimi 동기화) · 교재/진도 · 과제 · 텍스트 일괄 입력(파싱) · 학부모 리포트(카톡 발송) · 수강료 기록 · PC/패드/폰 사용 · 월 0원 인프라.

## 3. Non-Goals (MVP 제외)

- 학생/학부모 **등록·수정 UI** (GoAlimi가 마스터 — 조회만)
- GoAlimi 기능 흡수·화면 임베드 (분리 운영 확정 + iframe 불가 — 11_STUDY. 새 탭 런처만 제공, REQ-910)
- 회계·급여·미납 추적·월정액 관리 (수강료는 단순 기록만)
- 상담 CRM, 시험/성적 관리, LMS, 화상수업, 다지점
- 완전한 오프라인 동기화 (입력값 보존 수준만)
- 학부모용 로그인/포털 (리포트는 카톡 수신만)

## 4. Users

- Primary: 원장(owner), 강사(teacher) — 파일럿에서는 권한 동일, 역할 표시만
- Secondary: 학부모 (카톡 수신자, 시스템 미접속)

## 5. User Journey (요약 — 상세는 02_USER_FLOW)

- 수업: 오늘 화면 → 학생 탭 → [수업 시작] → 진도 저장 → 과제+코멘트 저장 → [완료]
- 일괄 입력: 텍스트 붙여넣기 → 파싱 미리보기 → 수정 → 일괄 저장
- 리포트: 기간·학생 선택 → 자동 생성 → 검토·수정 → [발송] → Bridge → 카톡
- 수강료: [결제 입력] → 학생·과목별 금액·수단 → 저장
